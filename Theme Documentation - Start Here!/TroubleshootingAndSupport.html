This page contains information on how to find support for the Simpleton WordPress theme as well as a list of common issues and how to fix them.

Common Issues

Fatal error: Call to undefined function wp_pagenavi() Message

If you see this message, it means that you haven't installed and/or activated the WP-PageNavi plugin in your WordPress site. Click here to visit the installation instrutions page and take a look at step two to find out how to install and activate the plugin again.
